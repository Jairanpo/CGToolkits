Asset creation

The asset tab will help you to create an asset inside an "AssetLibrary" or "Asset" folder structure.
Each asset is agUIoused of eight folders:

  • 01_Artwork: Creative department guidelines for the asset creation.
  • 02_Modeling: Modeling department folder containing the 3D object.
  • 03_Rigging: This folder will contain the animatable structure for the asset.
  • 04_Textures: Here we store the 2D texture that will be applied to the object.
  • 05_Shading: Surface information for the object
  • 06_Animation: Here will be the stored libraries or controller pickers that animators might need to setup.
  • 07_FX: FX simulations that are not dependant on a shot, or pre-build houdini networks
  • 08_Cache: FX templates in alembic format.
